# MediaHub Metadata Editor

## Version 0.3.0

Der Metadata Editor ist der zentrale lokale Medienbrowser für MediaHub. Die neue dreigeteilte Oberfläche zeigt Navigation, Bibliothek und Editor gleichzeitig und bleibt vollständig im Heimnetz.

### Funktionen

- Explorer-Navigation für alle Medien, Kanäle, Serien, Playlists und Entwürfe
- Suche und Gruppenfilter
- vollständiges Metadatenformular
- Live-Vergleich während der Eingabe
- Entwürfe speichern, laden und löschen
- NFO-Dateien UTF-8-sicher erstellen oder aktualisieren
- Poster, Fanart, Banner und Thumbnail austauschen
- automatische Sicherungen vor Dateiänderungen
- kontrollierte MediaHub-Schreibaktion, sobald die Hauptprogramm-API verfügbar ist

Die Oberfläche ist unter `/metadata-editor` erreichbar.


## Native MediaHub-Oberfläche

Ab v0.3.6 kann der Metadata Editor direkt im MediaHub-Hauptfenster eingebettet werden. Die lokale Weboberfläche bleibt zusätzlich verfügbar.


## Desktop-Fenster

Der Metadata Editor wird über „Plugin-Oberflächen“ als eigenes, frei vergrößerbares Desktop-Fenster geöffnet. WebRemote und Mobile Dashboard werden dort nicht angezeigt.
